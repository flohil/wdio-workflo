Wdio-Workflo

A Framework for writing Functional System Tests in TypeScript

Wdio-workflo helps you write functional system tests for web applications. It increases the reusability, maintainability and analysability of your tests. The framework is based on, extends and customizes the fabulous webdriverio-v4.

Getting Started

Learn how to install, configure and use wdio-workflo in the framework's Getting Started guide.
