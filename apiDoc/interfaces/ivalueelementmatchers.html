Interface IValueElementMatchers<ValueType>

This interface describes positive and negative (.not) expectation matchers for ValuePageElements.

It is implemented by the return value of the expectElement function if expectElement was passed an instance of ValuePageElement.

template

ValueType the type of the values handled by this elements' xxxValue functions
