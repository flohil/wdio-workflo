Interface IValueMapMatchers<K, ValueType>

This interface describes positive and negative (.not) expectation matchers for ValuePageElementMaps.

It is implemented by the return value of the expectMap function if expectMap was passed an instance of ValuePageElementMap.

template

K the names of the elements stored in the map (the map's keys) as string literals

template

ValueType the type of the values handled by the map's elements' xxxValue functions

Type parameters

  • K: string | number | symbol
